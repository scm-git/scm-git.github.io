## Install IDEA on Ubuntu
1. Install Ubuntu Make
```
sudo add-apt-repository ppa:ubuntu-desktop/ubuntu-make
sudo apt-get update
sudo apt-get install ubuntu-make
```
2. Install IDEA
```
sudo umake ide idea-ultimate
```
3. Install WebStorm
```
sudo umake ide webstorm
```
