云管理平台(CMP)

项目描述：

CMP是一个云服务管理平台，通过该平台，资源提供方可以将资源接入到Fox Cloud 平台中，资源购买方可以通过该平台购买云计算资源；买卖双方都可以通过该平台管理自己的资源，包括云主机CPU, 内存，硬盘，网络等。 系统采用微服务架构，总共分了20+个微服务模块，所有微服务模块将自己注册到Spring Cloud Eureka Discovery 服务器上，模块之间调用全部使用Spring Cloud 的Feign Client 方式，系统使用Netflix Zuul 作为路由服务，使用OAuth2 + Spring Security 做安全校验；其中有三个模块暴露给前端AngularJS，其他模块全部作为这三个模块的后端模块提供服务。部署方面，所有模块全部通过maven 插件以Docker 镜像的方式打包，并部署到Kubernetes 环境。

责任描述:

  • 负责用户中心微服务模块，实现用户中心所有接口，包括部门/机构管理相关接口，用户管理相关接口等。
  • 负责AWS 平台接入模块，完成AWS EC2，IAM 等相关资源的操作接口
  • 负责阿里云平台的维护工作，bug 修复等
  • 提供其他微服务模块需要的接口，包括部门配额管理接口（增加/减少陪，更新已使用额度等），根据企业/部门ID 获取所属用户等
  • 指导初级工程师完成相关功能模块开发，并解决他们遇到的问题

责任描述:
